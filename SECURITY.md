# Security Policy

NXS-Go is currently a local-first Pygame prototype. It does not require accounts, cloud services, telemetry, or network access.

## Reporting Issues

Please open a GitHub issue for security or privacy concerns.

## Current Security Principles

- Keep gameplay local-first.
- Do not require API keys or external credentials to run the game.
- Do not collect player data.
- Keep saved session history local to the player's machine.
- Review generated history files before sharing them publicly.
